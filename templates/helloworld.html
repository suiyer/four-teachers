Running Locally

To run your new app locally, you need to install a few dependencies. I highly recommend you use virtualenvwrapper to isolate your projects. With or without virtualenvwrapper, run the command below to install the required dependencies: 

$ pip install -r deployment/requirements.txt

Once you've installed the dependencies, use the following command to start the tornado server locally: 

$ python server.py -debug

Your app should be accessible at http://localhost:8888/hello.

Deploying Changes

Deploying your changes is as easy as running the following command: 

$ pushChanges

pushChanges will package up your web app and push it out to your EC2 instance.
